Schematics

AngularのSchematicsという仕組みで、いくつかのコンポーネント雛形が提供されている。

# 普通に空のコンポーネントを生成する場合
$ ng generate component <出力先>

# Angular Materialのコンポーネント雛形を生成する場合
$ ng generate @angular/material:<Schematic名> <出力先>

以下、提供されている雛形をそれぞれ説明する。

navigation

画面上部のバーと左側メニューの雛形。このサンプルそのものを作成するのに使用した。

address-form

配送先入力フォーム。

dashboard

画像を並べて表示する。

table

ソートとページング機能を持つテーブル。

tree

ディレクトリ等のツリー構造を表現する。
